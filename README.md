## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running Examples

```bash
# Run in development mode
npm run dev

# Or build and run
npm run build
npm start
```

### Running Tests

```bash
npm test
```
